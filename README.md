# SEN 401 Software Configuration Management Lab

## Overview

This project is a simple Student Management application developed for the SEN 401 Software Configuration Management and Maintenance laboratory assessment.

The application demonstrates the use of Git, GitHub, version control, branching, and modular Python programming.

## Project Structure

```
SEN-401-lab/
│
├── app.py
├── students.py
├── requirements.txt
├── README.md
│
└── utils/
    └── helpers.py
```

## Files

- **app.py** - Main program that displays the student report.
- **students.py** - Stores student names and scores.
- **utils/helpers.py** - Contains helper functions for processing student data.
- **requirements.txt** - Lists project dependencies.
- **README.md** - Project documentation.

## How to Run

```bash
python app.py
```
